## Getting Started

To get a local copy up and running follow these simple example steps.
```
    git clone git@github.com:LoboArkano/crypto-scope.git
```

### Prerequisites

- npm

- Node.js

### Setup

Install dependencies with:

```
    npm install
```

Start app with:

```
    npm start
```

Run test with:

```
    npm test
```
